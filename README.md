# Faithtop
Go语言的桌面GUI框架,用于快速编写桌面应用(使用[Qt](https://github.com/therecipe/qt)实现)

## [文档](https://github.com/gofaith/faithtop/wiki)

## 快速开始

```shell
qtdeploy -tags=impl build desktop
```
